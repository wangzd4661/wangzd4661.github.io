总结

  1. LinkedList基于链表实现，有序列表，可以插入null和重复元素。平均情况下，增删优于ArrayList，查询逊于ArrayList，非线程安全。
  2. 遍历需要使用Iterator方式或者foreach方式遍历。特别在元素量很大时，不能选择随机访问方式。
  3. 需要使用队列或者栈等数据结构时，可以选择LinkedList
